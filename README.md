# AI SEO Blog & Content Cluster Generator

## Project Overview

This project demonstrates how AI and Prompt Engineering can be used to create SEO-focused content strategies for local business websites.

The goal is to generate ranking-focused blog content, content clusters and local SEO strategies.

---

## Business Selected

### GlowNest Salon - Hyderabad

Business Type:
Unisex Salon

Target Audience:
Customers searching for salon services in Hyderabad

---

## Main SEO Topic

Best Salon Services in Hyderabad

Primary Keyword:

best salon in Hyderabad

---

## Problem

Many businesses publish random blogs without SEO strategy.

They struggle with:

- Google rankings
- keyword targeting
- structured content
- local search visibility

---

## Solution

Created a reusable AI SEO prompt workflow that generates:

- Pillar blogs
- Supporting blog clusters
- Keyword strategies
- SEO outlines
- Local SEO content

---

## Prompt Logic

The prompts use:

- Business information
- Location
- Target audience
- Search intent
- SEO structure

to create helpful and business-focused content.
